# Password Generator

This Python script is an interactive password generator. It allows you to specify a word for the beginning, a word for the end, and the number of characters between the two words. You can also choose to include uppercase, lowercase, numbers, and special characters in the central part of the password.

## Usage

1. Run the script using Python: `python3 passwd_en.py`
2. Follow the instructions to specify the starting and ending words, the number of characters between them, and the desired characters.
3. The script will generate the password and display it. It will also save the starting and ending words in the 'start.txt' and 'end.txt' files for future use.

## Generating Multiple Passwords

If the "password.txt" file already exists, the script will create a new file with a numeric suffix to avoid overwriting the existing file.

# Générateur de Mot de Passe

Ce script en Python est un générateur de mot de passe interactif. Il vous permet de spécifier un mot pour le début, un mot pour la fin, et le nombre de caractères entre les deux mots. Vous pouvez également choisir d'inclure des majuscules, des minuscules, des chiffres et des caractères spéciaux dans la partie centrale du mot de passe.

## Utilisation

1. Exécutez le script en utilisant Python : `python3 passwd_fr.py`
2. Suivez les instructions pour spécifier les mots de début et de fin, le nombre de caractères entre les deux, et les caractères souhaités.
3. Le script générera le mot de passe et l'affichera. Il enregistrera également les mots de début et de fin dans les fichiers 'debut.txt' et 'fin.txt' pour une utilisation future.

## Génération de plusieurs mots de passe

Si le fichier "mot_de_passe.txt" existe déjà, le script créera un nouveau fichier avec un suffixe numérique pour éviter l'écrasement du fichier existant.
